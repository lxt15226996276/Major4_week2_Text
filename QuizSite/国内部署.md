# 国内部署指南（Cloudflare 打不开时用）

`workers.dev` / `pages.dev` 在国内很多网络**无法访问**，这不是你操作错了。

下面两个方案**免费**，且**电脑关机也能用**。

---

## 方案 A：码云 Gitee Pages（最推荐，国内快）

### 第一次

1. 双击 **`上传码云.bat`**（会打包并打开 `dist` 文件夹）
2. 注册 https://gitee.com（手机号即可）
3. **新建仓库**
   - 名称：`major4-quiz`
   - 类型：**公开**
4. 进入仓库 → **上传文件** → 把 `dist` 里的**所有文件和文件夹**上传  
   （根目录要有 `index.html`）
5. 仓库页 → **服务** → **Gitee Pages** → **启动**
   - 分支：`master`（或 `main`）
   - 目录：`/`（根目录）
6. 等待 1～2 分钟，得到永久链接：

```
https://你的Gitee用户名.gitee.io/major4-quiz/
```

### 更新题库

改 Excel → 双击 **`上传码云.bat`** → 在 Gitee 仓库覆盖上传 → Pages 重新发布。

---

## 方案 B：GitHub Pages（备选）

1. 安装 Git + 注册 GitHub
2. 双击 **`部署到公网.bat`**
3. Settings → Pages → 分支 `gh-pages`
