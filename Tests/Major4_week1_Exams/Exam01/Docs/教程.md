**登录凭据（试卷规定）：**
- 账号 = 考生姓名全拼
- 密码 = `888888`
- 成功：控制台输出「登陆成功」→ 跳转服务器场景
- 失败：控制台输出「登录失败」

## 场景清单

| 场景名 | 路径 | 说明 |

| LoginScene | `Scenes/LoginScene.unity` | 登录界面 |

## 脚本清单

| 脚本名 | 挂载对象 | 职责 |
|--------|----------|------|
| `LoginController.cs` | LoginCanvas 或 GameManager | 读取账号密码、校验、跳转 |
| `ServerListController.cs` | ServerCanvas | 动态生成 10 条服务器、选中逻辑 |
| `ServerItem.cs`（可选） | ServerItem 预制体 | 单条服务器 UI 的选中状态 |
