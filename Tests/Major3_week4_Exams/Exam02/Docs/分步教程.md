# Exam02 分步教程 · 宝箱开启 + 视频 + 特效

> 三项合计 30 分，按顺序做即可。

---

## 前置检查

| 项目 | 要求 |
|------|------|
| `Box` | 有 **Box Collider**，勾选 **Is Trigger** |
| `Player` | Tag = **Player**，有 **CharacterController** |
| 视频 | `Assets/Exams/Video/Testvideo.mp4`（项目里已有） |
| 特效 | `Assets/Exams/Exam02/Effects/TreasureGlow.prefab` |

---

## Step 1 · 碰到宝箱触发开启事件（10分）

### 1.1 挂脚本

1. 选中场景里的 **Box**
2. Add Component → `TreasureChestController`（命名空间 Exam.Exam02）

### 1.2 确认碰撞设置

- **Box**：`Is Trigger` ✅
- **Player**：Tag = `Player` ✅

### 1.3 核心代码（已写好，理解即可）

```csharp
void OnTriggerEnter(Collider other)
{
    if (opened) return;
    if (!other.CompareTag("Player")) return;  // 必须是玩家
    opened = true;
    OpenTreasure();  // 触发后续流程
}
```

### 1.4 测试

运行游戏，WASD 走到 Box 旁边，Console 应出现 **「宝箱开启！」**。

---

## Step 2 · 屏幕播放宝藏视频（10分）

### 2.1 创建视频播放器

场景中已有 **`TreasureVideoPlayer`** 空物体（若没有，自己建）：

1. Hierarchy 右键 → Create Empty → 命名 `TreasureVideoPlayer`
2. Add Component → **Video Player**
3. 设置：

| 属性 | 值 |
|------|-----|
| Video Clip | `Testvideo`（拖 `Assets/Exams/Video/Testvideo.mp4`） |
| Render Mode | **Camera Far Plane**（全屏铺在相机上） |
| Camera | 拖入 **Main Camera** |
| Play On Awake | ❌ 取消 |
| Loop | ❌ 取消 |

### 2.2 关联到宝箱脚本

选中 **Box** → `TreasureChestController`：

- **Video Player** 拖入 `TreasureVideoPlayer` 上的 Video Player 组件

### 2.3 流程

```
玩家碰到 Box → videoPlayer.Play() → 屏幕全屏播视频
```

---

## Step 3 · 视频结束后生成特效（10分）

### 3.1 特效预制体

使用 `Assets/Exams/Exam02/Effects/TreasureGlow.prefab`（金色传送光芒）。

### 3.2 关联

选中 **Box** → `TreasureChestController`：

- **Effect Prefab** 拖入 `TreasureGlow`

### 3.3 核心代码

```csharp
void OnVideoFinished(VideoPlayer vp)
{
    videoPlayer.Stop();
    SpawnEffect();  // 视频播完才生成
}

void SpawnEffect()
{
    Instantiate(effectPrefab, transform.position, Quaternion.identity);
}
```

### 3.4 完整流程自测

1. ▶ 运行
2. 走到 Box → 屏幕播视频
3. 视频结束 → Box 位置出现光芒粒子特效

## 常见问题

| 现象 | 解决 |
|------|------|
| 碰到没反应 | Box 没勾 Is Trigger；Player Tag 不对 |
| 视频不播 | Video Player 没拖；Clip 没指定；Camera 没指定 |
| 没特效 | Effect Prefab 没拖；视频没播完（Loop 要关掉） |
| 触发多次 | 脚本里 `opened` 已防重复，检查是否挂了多个脚本 |
