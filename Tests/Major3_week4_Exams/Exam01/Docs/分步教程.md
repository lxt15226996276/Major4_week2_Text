## Step 1 · 创建两个场景 + game1 基础环境 ✅ 当前步骤

### 目标
- 在 `Exam01/Scenes/` 下创建 `game1.unity`、`game2.unity`
- game1 放：Camera、Light、Ground(Cube)、空物体 `SceneRoot`
- 验证场景能保存、Build Settings 能添加

### Inspector 操作（逐步做）

#### 1.1 创建 game1
1. 菜单 **File → New Scene** → 模板选 **Basic (Built-in)** 或 **3D**
2. **File → Save As** → 路径：`Assets/Exams/Exam01/Scenes/game1.unity`

#### 1.2 调整 Main Camera
选中 **Main Camera**，Inspector：

| 属性 | 推荐值 | 含义 |
|------|--------|------|
| Transform Position | (0, 1, -10) | 相机位置 |
| Transform Rotation | (0, 0, 0) | 朝向 |
| Camera → Clear Flags | Solid Color | 清屏方式 |
| Camera → Background | #333333 或黑色 | 背景色 |
| Camera → Field of View | 60 | 透视角度 |

#### 1.3 调整 Directional Light
| 属性 | 推荐值 |
|------|--------|
| Rotation | (50, -30, 0) |
| Intensity | 1 |
| Color | 白色 |

#### 1.4 创建 Ground
1. 右键 Hierarchy → **3D Object → Cube**
2. 重命名 `Ground`
3. Transform：

| 属性 | 值 | 为什么 |
|------|-----|--------|
| Position | (0, 0, 0) | 放原点 |
| Scale | (10, 1, 10) | 压扁成大平台 |

4. 组件 **Box Collider**：默认即可（Size 随 Scale 自动变）

#### 1.5 创建 SceneRoot 空物体
1. 右键 Hierarchy → **Create Empty**
2. 命名 `SceneRoot`（后面挂视频控制脚本用）
3. Position (0,0,0)

#### 1.6 创建 game2（空场景即可）
1. **File → New Scene → Basic (Built-in)**
2. **Save As** → `Assets/Exams/Exam01/Scenes/game2.unity`
3. 同样保留 Camera + Light + Ground（Scale 同上），后面 Step 3 再加 Player

#### 1.7 加入 Build Settings
1. **File → Build Settings**
2. 把 `game1`、`game2` 拖进 **Scenes In Build**
3. **game1 必须在 Index 0**（第一行），game2 在 Index 1  
   - 原因：后续 `LoadScene("game2")` 或按名称加载都依赖场景已注册

### 本步完成标准
- [ ] Project 窗口能看到两个场景文件
- [ ] Build Settings 里 game1=0，game2=1
- [ ] 打开 game1 Play 无报错（只有空场景）
